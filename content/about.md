Hur sidan är uppbyggd
==================

I kursmoment 2 har jag fått använda mig utav en hel del nya sätt att skriva sin css kod på.

Framförallt har vi fått mycket användning utav SASS. SASS är en "utökning" utav CSS som har många nya funktioner som CSS inte erbjuder.

Bland annat har jag använt mig utav variablar för att ersätta färger, detta förenklar om man senare vill ändra temat så slipper man leta runt bland alla olika css filer.

Det leder mig till den andra delen av SASS som vi har använt där vi använder så mycket SCSS filer som möjligt för att dela upp våra bitar i sektioner. I sin tur leder detta till att 

det blir enklare att hitta när man behöver ändra i sitt tema.

Till sist så har vi en fil som kompilerar alla våra SCSS filer till vanliga CSS en vanlig CSS fil som sedan läses in på våran webbplats.
